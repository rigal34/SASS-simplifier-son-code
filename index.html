1er étape avant de commencer le SASS

code sans refactoring = sans organisation

différents exemple de boutons

.turquoise{ padding: 15px; border: none; width: 200px; background: #48D1CC; }
.round{ padding: 15px; border: none; width: 200px; background: #48D1CC; border-radius: 30px; color: #242424; }
.wide{ padding: 15px; border: none; width: 400px; background: #48D1CC; color: #242424; }
.orange-round{ padding: 15px; border: none; width: 200px; background: orange; border-radius: 30px; }

j'applique le DRY(Do not repeat yourself)

J'enléve tout ce qui peut-être répetitif dans mon code


Pour en faire un générique commun


.btn-round { border-radius: 30px; }

RECFACTORING + DRY(Do not repeat yourself)

.wide { padding: 15px; width: 400px; border: none; color: #242424; background: #48d1cc; }
.orange-round { padding: 15px; border: none; width: 200px; background: orange; border-radius: 30px; }
